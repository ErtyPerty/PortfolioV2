Box model and outlines

Dotted outline defines an outline with cool dots, it's simillar to the dashed outline style, however the dots are smaller and more frequent than the dash

A dashed outline is simillar to a dotted outline, but instead of using dots, it uses larger and less frequent dashes

A solid outline makes a single rectangle around the content

A double outline makes 2 rectangles around the content

A groove outline makes the rectangle look cool by adding a shadow going through the outline to give it an effect of 3d

Ridge gives the effect of 3d to the outline by shading the outside edges of the outline to make it appear like it has a shadow

Gives an inverted, inwards facing 3d effect to the outline

Opposite to the inset outline, this outline gives an outwards facing 3d effect to the outline

Anchor links

Anchor links in HTML are used to navigate through a website, either navigating on a page or navigating to a different page in the website, anchor links can do all that! It works by first giving it the tag "a", then it takes an href which is the ID of a section in a page, or a link to a different web page. After these 2 inputs, a link will generate, and like most HTML things, you can customise the link. When you click on the link, you will be taken to wherever the ID specifies. One example of this is the dropdown box on the top left of the screen and the links at the top. The drop down box uses ID's to navigate a page, whereas the links use anchor links to navigate to different pages, go on, give it a try!
